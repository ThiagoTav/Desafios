História do Computado

voltar

A história dos computadores são divididos em 5 Gerações sendo o começo na 1ªGeração: O primeiro computador foi criado em 1944 e se chava mark 1, foi criado para calcular tabelas de trajetória para fazer a pontaria de canhões de longo alcance, substituindo os demorados cálculos à mão. O segundo foi criado em 1946 e se chamava colossus foi criada para ajudar a decifrar mensagens de rádio teletipo interceptadas que foram criptografadas. O terceiro foi criado em 1946 se chava ENIAC foi criado para realizar cálculos complexos dos laboratórios de testes de balística.

2ªGeração(1959 a 1965) de computadores foi marcado pela criação do transistores, um dispositivo semicondutor criado com o objetivo de trocar ou amplificar a potência elétrica e também os sinais eletrônicos. 3ªGeração(1965 a 1970) marcado pela criação do circuito integrado, transistor,capacitores e resistores. 4ªGeração(a partir de 1971) 5ªGeração marcado pela AI, PC quântico e cripto moedas.

O que são Clients?

voltar

É quando você consome uma informação da Internet, usando a internet como meio de comunicação com dispositivos com acesso a internet, quando você acessa o site ele vai na internet e pede os arquivos do site para o servidor e o servidor retorna essas informações.

O que são Servers?

voltar

Pode ser um Software ou um Computador que fornece serviços para uma rede de computadores chamada Cliente, o servidor é um computador mais robusto porque existe muitos clientes conectados neles, servidores armazenam arquivos do site para que o cliente consiga se conectar com seu site, e existem tipos de servidor.

Linguagens de Programação

voltar

É uma liguagem formal que especifica um conjunto de instruções e regras usadas para gerar programas, um conjunto de tags para que o computador consiga ler para aplicar as mudanças.Linguagens mais populares são: Javascript, Python, Java, PHP e C#.

Como é uma estrutura básica de um HTML?

voltar

Toda tag tem uma "<" e uma palavra que seria a tag e ">", as basicas são < html >, < head >, < title > e < body >, essas são as tags basicas do HTML. Lembrando que quando abrir uma tag você precisa fechar com o "/" e o nome da tag.

Entendendo a diferença entre listas ordenadas e não ordenadas

voltar

A diferença entre a lista ordenada e a lista não-ordenada é: a lista ordenada enumera os elementos, a lista não-ordenada coloca marcadores antes de cada elemento.
